# Board

[![CircleCI](https://circleci.com/gh/retroboard/board/tree/master.svg?style=svg)](https://circleci.com/gh/retroboard/board/tree/master)

### Initial Setup

Requirements:
  - npm
  - node >~ v10.11.0

```
$ npm install
```

### Running locally

```
$ npm start
```

### Running the tests

```
$ npm test
```
